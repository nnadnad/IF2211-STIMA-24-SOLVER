Program ini adalah program solver untuk game kartu 24 dengan algoritma greedy

Untuk menjalankan GUI, buka file gui_batch17_kelompok22.py pada folder main

Untuk menjalankan dengan file eksternal, buka file batch17_kelompok22.py AA BB dengan
AA file input dan BB file output. Jangan lupa ekstensinya

Folder resources adalah folder untuk menyimpan semua gambar yang digunakan pada GUI
